# wx_shopping_mall
### 前言
之前公司一直都用vue做项目，最近换了一家做小程序的，便利用这几天好好恶补了一下，发现小程序与vue真是极其相似，都是数据驱动、组件化开发，也定义了一下类似v-if、v-for之类的指令，感觉做过vue的简直可以无缝切换到小程序开发。
